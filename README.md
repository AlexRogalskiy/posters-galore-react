
# Posters Galore

Posters Galore using react

## Installation

```
make install

```

## Start

Start the demo

```
make start
```

## Start

Stop the demo

```
make stop
```

## Test

Run tests of the application

```
make test
```

## Test e2e run

Run e2e tests (heahdless mode)

```
make cypress-run
```

## Test e2e watch

Run e2e tests

```
make cypress-open
```
